Walkthrough

This Project Contains the following items


Numeric Method - I

Contains the toExponential() method, which forces a number to be displayed as exponential even though it lies in the normal range of Javascript Numbers.


Numeric Method - II

Contains the toPrecision() method, which defines how many total digits (including digits to the left and right of the decimal) to display of a number


String Method - I

Contains the charAt() method, which returns the character at the specefied index


String Method - II

Contains the substr() method, which Returns the characters in a string beginning at the specified location through the specified number of characters.


String Wrapper Method - I

Contains the fontcolor() method, which Causes a string to be displayed in the specified color as if it were in a <font color="color"> tag.


String Wrapper Method - II

Contains the fontsize() method, which Causes a string to be displayed in the specified size as if it were in a <font size="size"> tag.


Array Method - I

Contains the reverse() method, which Reverses the order of the elements of an array -- the first becomes the last, and the last becomes the first.


Array Method - II

Contains the sort() method, which Sorts the elements of an array.


Date Method - I

Contains the Date() method, which Returns today's date and time.


Date Method - II

Contains the getDay() method, which Returns the day of the week for the specified date according to local time.


Math Method - I

Contains the abs() method, which Returns the absolute value of a number.


Math Method - II

Contains the sqrt() method, which Returns the square root value of a number.


Login And Register

Demonstrates the Login and Register Functionality using localstorage capabilities in javascript.
